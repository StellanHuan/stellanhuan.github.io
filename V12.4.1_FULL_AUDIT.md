## 二、官网数据读取

### 数据流图
```
Admin Sections 编辑
  │ deploy() → GitHub API PUT
  ▼
data.js + series.js + site.json (GitHub repo)
  │ CI: export_content.py
  ▼
data.js (regenerated from site.json sections)
  │ GitHub Pages deploy
  ▼
stellanhuan.github.io
  │ <script src="data.js?v=6">
  ▼
Browser: window.SITE_DATA
```

### 数据源
| 文件 | 加载方 | 内容 |
|------|--------|------|
| data.js | index.html | SITE_DATA {photos, sections} |
| src/data/series.js | series/index.html | SERIES_DATA {portrait,street,...} |
| data.json | - | 备份（CI 生成） |

### 重复数据源
- `data.js` 和 `src/data/series.js` 有重叠（photos 数组），但用途不同
- `data.json` 与 `data.js` 内容一致（CI 同步）
- 无冲突 ✅
