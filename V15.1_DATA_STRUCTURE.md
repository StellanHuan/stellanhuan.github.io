# V15.1 数据结构设计

## 1. site.json 图片字段规范

```json
{
  "photos": [{
    "file": "images/web_001.jpeg",
    "title": "光の粒子",
    "series": "portrait",
    "tags": ["人像", "自然光"],
    "orientation": "landscape",
    "display": "large",
    "camera": "NIKON Z 8",
    "location": "郑州",
    "date": "2026-06"
  }]
}
```

### 新增字段
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| orientation | enum | ✅ | landscape / portrait / square |
| display | enum | ✅ | large / medium / small |

## 2. orientation 枚举

| 值 | 判断规则 | 示例 |
|----|---------|------|
| landscape | width > height | 横版风光、街拍 |
| portrait | height > width | 竖版人像 |
| square | width ≈ height (差<5%) | 方构图 |

### 自动推断（CMS导入时）
```js
if (Math.abs(w - h) / Math.max(w, h) < 0.05) orientation = 'square'
else if (w > h) orientation = 'landscape'
else orientation = 'portrait'
```

## 3. display 枚举

| 值 | 含义 | 分配规则 |
|----|------|---------|
| large | 首图/封面级 | 每个 section 前 1-2 张 |
| medium | 标准展示 | 默认值 |
| small | 辅助/补充 | 密集排列时使用 |

### CMS 默认值
- featured 第1张 → display: large
- 其余 → display: medium
- 用户可手动调整

## 4. PC 展示规则

### Masonry Grid 定位
```css
[data-orientation="landscape"][data-display="large"]  { grid-column: span 2; grid-row: span 2 }
[data-orientation="landscape"][data-display="medium"] { grid-column: span 2; grid-row: span 1 }
[data-orientation="portrait"][data-display="large"]   { grid-column: span 1; grid-row: span 2 }
[data-orientation="portrait"][data-display="medium"]  { grid-column: span 1; grid-row: span 1 }
[data-orientation="square"]                            { grid-column: span 1; grid-row: span 1 }

/* 图片容器比例 */
[data-orientation="landscape"] img { aspect-ratio: 4/3 }
[data-orientation="portrait"] img  { aspect-ratio: 3/4 }
[data-orientation="square"] img    { aspect-ratio: 1/1 }
```

### 删除的 CSS
- `.masonry>:nth-child(odd) img { aspect-ratio: 3/4 }` ❌
- `.masonry>:nth-child(even) img { aspect-ratio: 4/3 }` ❌
- `.masonry>:first-child img { aspect-ratio: 4/5 }` ❌
- 所有 `:nth-child()` 图片比例规则 ❌

## 5. Mobile 展示规则

```css
@media(max-width:768px) {
  /* 所有图片全宽，根据 orientation 设定比例 */
  .masonry .duo-item img[data-orientation="landscape"] { aspect-ratio: 16/9 }
  .masonry .duo-item img[data-orientation="portrait"]  { aspect-ratio: 3/4 }
  .masonry .duo-item img[data-orientation="square"]    { aspect-ratio: 1/1; width: 85%; margin: 0 auto }
  .masonry .duo-item img[data-display="large"]         { aspect-ratio: 4/5 }
}
```

## 6. 从旧 CMS 迁移方案

### 迁移脚本（一次性）
```python
for photo in photos:
    if not photo.get('orientation'):
        # 从图片文件读取实际尺寸
        w, h = get_image_size(photo['file'])
        photo['orientation'] = 'landscape' if w > h else 'portrait' if h > w else 'square'
    if not photo.get('display'):
        photo['display'] = 'medium'
```

### CMS 升级

- Admin → Photos 页面增加 display 下拉框
- 批量操作：选中多张 → 批量设置 orientation/display
- Sections 编辑器：拖入照片时保留 orientation 元数据
